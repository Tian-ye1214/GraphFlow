## 组 1（Agent 1）— 后端 P0 使能（无前端）
**Files**：`backend/app/routers/runs.py`、`backend/app/routers/workflows.py`、`backend/app/routers/datasets.py`、`backend/app/services/*`、`backend/tests/`(新增)

1. **运行耗时**：`_run_out`(runs.py) 增加 `started_at`（Run 模型已有？若无则用 created_at 起算；优先暴露已有时间戳）。确保 RunsPage/RunDetail 能算时长。
2. **单节点重跑**：`POST /api/runs/{id}/rerun-failed` 增加可选 body/query `node_id`：仅重置该节点及其下游 descendants 的失败行（descendants 逻辑已存在，把入参从全集改单节点）；不传则维持现「全部失败节点」行为。
3. **复制工作流**：新增 `POST /api/workflows/{id}/duplicate` → 复用 workflow_package 的 collect/克隆能力（或直接复制 graph_json + 引用），返回新建工作流（名「原名 副本」/递增），归属当前用户，校验 _get_owned。
4. **示例数据集**：新增 `POST /api/datasets/sample` → 把 1-2 份内置小样本（如 20 行中文短句 q 列、20 行待分类评论）注入当前租户（走 create_dataset_from_upload 或直接建分片数据集），返回新数据集列表。样本内容内置在后端常量/随包小文件。
- 每项 TDD 加测试；`pytest -q` 全绿。**不动前端**。
